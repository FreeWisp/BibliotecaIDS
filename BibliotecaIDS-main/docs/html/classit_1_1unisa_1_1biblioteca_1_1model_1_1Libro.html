Detailed Description

Rappresenta un libro con le sue informazioni principali.

Questa classe contiene tutti i dati relativi a un libro presente in biblioteca: informazioni bibliografiche (titolo, autore, ISBN, anno) e informazioni gestionali (numero di copie disponibili).
