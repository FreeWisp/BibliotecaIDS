Detailed Description

Repository per la gestione delle entità Prestito.

Fornisce le operazioni CRUD e ulteriori servizi dedicati al monitoraggio dello stato dei prestiti e al controllo delle regole del sistema, come il limite massimo di prestiti per utente.
